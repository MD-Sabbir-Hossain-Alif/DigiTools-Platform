# ✨ DigiTools-Platform

**Digital Tools Platform** built with modern frontend technologies. (using React)

![Project Banner](https://i.ibb.co.com/chwFk6m8/digitools-screenshot.png)

---

### 📖 **A Little Description**

DigiTools-Platform is a clean and modern web application that serves as a **design tools (like: ai tools)**. It helps users explore, learn about, and purchase digital design & content resources and tools.

This project was created by a **obsessed learner** to practice React.js, component-based development, responsive design, and state management. It features a professional landing-page style UI with navigation, pricing cards, and a working cart — perfect for just start learning React journey!

---

### 🛠 **Technologies Used**

| Technology         | Purpose                          | Why I Love It                               |
| ------------------ | -------------------------------- | ------------------------------------------- |
| **React.js**       | Frontend framework               | Component reusability & fast learning curve |
| **Vite**           | Build tool & development server  | Lightning-fast hot reloads                  |
| **Tailwind CSS**   | Styling framework                | Utility classes = super quick styling       |
| **DaisyUI**        | Tailwind component library       | Beautiful pre-built components              |
| **JavaScript ES6** | Logic & interactivity            | Core language I’m learning                  |
| **npm**            | React-Toastify                   | Easy dependency handling                    |
| **JSON**           | Structured human-readable format | Easy data handling & faster                 |

---

### 🌟 **3 Key Features**

1. **🛒 Working Shopping Cart**  
   Add design tools to your cart and see the total count instantly update in the navbar — basic state management in action!

2. **💰 Interactive Product Cards**  
   Beautifully designed pricing plans with and call-to-action buttons.

3. **🚀 Responsive Web Page**  
   Clean navbar with smooth mobile menu, logo, links, and a **live product cart counter** that updates in real-time.
