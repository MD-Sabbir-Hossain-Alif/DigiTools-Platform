# ✨ DigiTools-Platform

**A beautiful Design Tools Platform (Ai)** built with modern frontend technologies. (using React)

![Project Banner](https://i.ibb.co.com/chwFk6m8/digitools-screenshot.png)

---

### 📖 **A Little Description**

DigiTools-Platform is a clean and modern web application that serves as a **design tools (like ai)**. It helps users explore, learn about, and purchase digital design resources and tools.

This project was created by a **beginner learner** to practice React.js, component-based development, responsive design, and state management. It features a professional landing-page style UI with navigation, pricing cards, and a working shopping cart — perfect for anyone just starting their React journey!

---

### 🛠 **Technologies Used**

| Technology       | Purpose                         | Why I Love It                                           |
| ---------------- | ------------------------------- | ------------------------------------------------------- |
| **React.js**     | Frontend framework              | Component reusability & fast learning curve             |
| **Vite**         | Build tool & development server | Lightning-fast hot reloads                              |
| **Tailwind CSS** | Styling framework               | Utility classes = super quick styling                   |
| **DaisyUI**      | Tailwind component library      | Beautiful pre-built components (no more CSS headaches!) |
| **JavaScript**   | Logic & interactivity           | Core language I’m learning                              |
| **npm**          | Package manager                 | Easy dependency handling                                |

---

### 🌟 **3 Key Features**

1. **🚀 Responsive Web Page**  
   Clean navbar with smooth mobile menu, logo, links, and a **live product cart counter** that updates in real-time.

2. **💰 Interactive Product Cards**  
   Three beautifully designed pricing plans (Basic, Pro, Enterprise) with hover effects and call-to-action buttons.

3. **🛒 Working Shopping Cart**  
   Add design tools to your cart and see the total count instantly update in the navbar — basic state management in action!
